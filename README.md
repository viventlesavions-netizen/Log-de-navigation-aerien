# Log de Navigation Aérien - Plan de Vol

**Plan de Vol** est une application web moderne conçue pour les pilotes, permettant de créer et de gérer des logs de navigation aéronautique de manière numérique avec exportation PDF. Cette solution remplace les traditionnels logs papier par une interface moderne et intuitive.

## Fonctionnalités

- **Gestion Numérique** : Saisie et organisation de vos paramètres de navigation.
- **Interface Intuitive** : Design épuré pour une lecture rapide, même en conditions de vol.
- **Multi-support** : Utilisable sur tablette ou ordinateur pour préparer et suivre votre navigation.
- **À propos** : Une section dédiée pour comprendre la genèse du projet et ses objectifs.

## Technologies utilisées

L'application repose sur des technologies web standards pour garantir légèreté et rapidité :

- **JavaScript** (46.0%) : Logique métier pour le calcul et la gestion interactive du log.
- **HTML** (34.0%) : Structure de l'application.
- **CSS** (20.0%) : Design personnalisé et responsive.

## Structure du projet

- `index.html` : Page d'accueil et point d'entrée.
- `lognav.html` : L'interface principale du log de navigation.
- `a-propos.html` : Informations sur le projet.
- `script.js` : Gestion des interactions et des données.
- `style.css` : Feuilles de style.
- `favicon/` : Ressources graphiques pour l'identité visuelle.
- `images/` : Illustrations et schémas (dont le DA40.png et avion.png).
